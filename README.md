## Prerequisites

* Ionic CLI `npm - ionic@beta --global`
* Cordova CLI

## Installation

```
npm install && ionic state restore
```

## Project public API

```
# Dev server
npm run start

# Dump files in www
npm run dumpdev
npm run dumpprod

# Run Cordova
npm run android
npm run ios
npm run iosEmulator

# Webpack bundle analyser 
npm run analyze
```
